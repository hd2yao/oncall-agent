# oncall-agent 后续优化方向

## 配置与安全
- 密钥与配置改为环境变量/Secret 管理，避免明文出现在配置文件
- 关键工具增加权限控制、调用审计与速率限制

## 记忆与检索
- 将内存记忆持久化至 Redis/DB，支持多实例与重启恢复
- 引入 reranker 或混合检索（关键词 + 向量）提升 RAG 质量
- 文档切分策略优化（动态分块、去噪与重复段落过滤）

## 可靠性与可观测性
- 标准化错误码与用户可读提示
- 增加 tracing、metrics（Prometheus）与链路可视化
- 引入 schema 版本与自动校验，避免旧数据导致检索失败

## 产品与体验
- 前端展示工具调用过程与引用文档
- 支持多会话/多知识库管理与切换
- 增加对话级配置（模型、温度、TopK）与调试面板
